普及冷门知识之曼努埃尔·佩莱格里尼

发表于 2017-05-16

谁是曼努埃尔？


曼努埃尔·佩莱格里尼（西班牙语：Manuel Luis Pellegrini Ripamonti，1953年9月16日－）生于智利圣地亚哥，前足球运动员，曾任西甲豪门皇家马德里主教练，现任中超联赛球队河北华夏幸福主教练。

智利工程师

在【百度知道】佩莱格里尼为什么叫工程师一问中，有答主答道：“他有智利大学工程系的毕业文凭”，这个答案并不准确。他却是是因为拥有土木工程师证书，才被昵称为“工程师”。但是，这里需要纠正的是，佩莱格里尼实际上毕业于智利天主教大学而并非智利大学。

为什么误解

1.佩莱格里尼首次效力的职业球队和执教的球队均为智利大学。
2.误认为智利大学俱乐部 = 智利大学的俱乐部。
3.无论是西班牙语还是英语维基百科均未填写他的受教育经历。
